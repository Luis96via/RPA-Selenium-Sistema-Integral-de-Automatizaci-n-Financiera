🎯 Descripción del Proyecto

Este es un sistema integral de automatización robótica (RPA) desarrollado con Python y Selenium que automatiza completamente el proceso de extracción, procesamiento y reporte de datos financieros desde un portal web.

🤖 RPA con Selenium

Automatización completa de login y extracción de datos de sitios web financieros

🔐 Resolución de Captchas

OCR personalizado para resolver captchas de texto, imagen y lógica

🌐 APIs Integradas

Integración con Frankfurter API para tasas de cambio en tiempo real

💾 Base de Datos MySQL

Almacenamiento estructurado y consultas SQL optimizadas

📊 Reportes Automatizados

Generación de reportes en Excel, XML y JSON con envío automático

🎨 Interfaz Web Moderna

Panel de control con terminal en tiempo real y documentación

🛠️ Stack Tecnológico

🤖 Automatización y Web Scraping

  • Python 3.x: Lenguaje principal del proyecto
  • Selenium WebDriver: Automatización de navegador web
  • ChromeDriver: Control del navegador Chrome
  • BeautifulSoup: Parsing de HTML (complementario)

🔐 Resolución de Captchas

  • OCR Personalizado: API REST Serverless desarrollada por el autor
  • Pillow (PIL): Procesamiento de imágenes
  • OpenCV: Análisis de imágenes para captchas

🌐 APIs y Servicios Externos

  • Frankfurter API: Tasas de cambio en tiempo real
  • Requests: Cliente HTTP para consumo de APIs
  • JSON: Procesamiento de respuestas de API

💾 Base de Datos y Almacenamiento

  • MySQL: Base de datos relacional principal
  • mysql-connector-python: Conector oficial de MySQL
  • Pandas: Manipulación y análisis de datos
  • CSV: Almacenamiento temporal de datos extraídos

📊 Generación de Reportes

  • OpenPyXL: Generación de reportes Excel
  • XML.etree.ElementTree: Generación de reportes XML
  • JSON: Generación de reportes JSON
  • Rich: Formateo de salida en consola (versión standalone)

📧 Envío de Emails

  • SMTP: Protocolo de envío de emails
  • EmailMessage: Construcción de mensajes
  • HTML Templates: Plantillas modernas para emails
  • python-dotenv: Gestión de variables de entorno

🌐 Interfaz Web

  • Flask: Framework web para la interfaz
  • HTML5/CSS3: Interfaz moderna y responsiva
  • JavaScript: Interactividad y polling en tiempo real
  • AJAX: Comunicación asíncrona con el servidor

🔧 Utilidades y Herramientas

  • Logging: Sistema de logs estructurado
  • Threading: Ejecución en hilos separados
  • Subprocess: Ejecución de procesos Python
  • Queue: Comunicación entre hilos

⭐ Características Destacadas

  • Automatización Completa: Proceso end-to-end sin intervención manual
  • Resolución Inteligente de Captchas: OCR personalizado para múltiples tipos de captcha
  • Integración de APIs: Consumo automático de tasas de cambio en tiempo real
  • Procesamiento de Datos: Conversión automática de monedas y normalización
  • Base de Datos Robusta: Creación automática de BD y tablas si no existen
  • Reportes Profesionales: Múltiples formatos (Excel, XML, JSON) con diseño moderno
  • Envío Automático de Emails: Plantillas HTML profesionales con adjuntos
  • Interfaz Web Moderna: Panel de control con terminal en tiempo real
  • Logging Detallado: Seguimiento completo del proceso con timestamps
  • Manejo de Errores: Sistema robusto de recuperación y notificación
  • Escalabilidad: Arquitectura modular y reutilizable
  • Documentación Completa: Guías técnicas y flujos de proceso
